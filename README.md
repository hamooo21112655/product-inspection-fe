# Product Inspection Frontend

Ovaj repozitorij sadrži frontend aplikaciju razvijenu u React i TypeScript tehnologijama, namijenjenu vizualnom prikazu, upravljanju i analizi podataka vezanih za inspekciju proizvoda.  
Aplikacija je fokusirana na modularnost, čitljivost koda i skalabilnost, uz korištenje modernih alata i provjerenih razvojnih praksi.

## Opis projekta

Cilj aplikacije je omogućiti:

- pregled i organizaciju inspekcijskih podataka,
- jasan i intuitivan korisnički interface,
- jednostavno proširenje funkcionalnosti (dashboardi, forme, izvještaji),
